# МережаЗв'язків • Теорія Графів & GNN

Простий та інтерактивний вебдодаток для візуалізації соціальних графів, аналізу спільних інтересів та формування рекомендацій за допомогою алгоритмів машинного навчання (Graph Convolutional Networks / GNN).

## 🚀 Що вміє цей проєкт?

* **Візуалізація графа (D3.js):** Інтерактивна мережа користувачів, де можна перетягувати вузли та переглядати зв'язки.
* **3D & AR/VR режим (Three.js):** Можливість переглянути граф у тривимірному просторі або в доповненій/віртуальній реальності (WebXR).
* **Розумні рекомендації:** Система аналізує спільні інтереси та топологію графа (Message Passing) для підбору потенційних друзів.
* **Редагування в реальному часі:** Можна додавати нових людей, обирати їхні інтереси та керувати зв'язками (створювати або розривати їх).

## 📁 Структура файлів

* `index.html` — головна сторінка сайту з інтерфейсом та вкладками.
* `styles.css` — стильне кіберпанк-оформлення з ефектом розмиття (Glassmorphic UI).
* `data.js` — генерація бази користувачів, пул інтересів та логіка ембедінгів GNN.
* `graph.js` — побудова 2D-графа та симуляція фізики за допомогою D3.js.
* `ui.js` — керування інтерфейсом, перемикання табів, додавання користувачів та інтеграція WebXR (3D/AR).
